# n8n - Company Research Report

## Company Overview

**n8n GmbH** (pronounced "n-eight-n") is a private workflow automation platform company based in Berlin, Germany. Founded in 2019 by Jan Oberhauser (CEO), n8n has become one of the fastest-growing automation companies in Europe, pioneering the "fair-code" licensing model and positioning itself as an AI-native workflow automation platform.

**Headquarters:** Berlin, Germany
**Founded:** 2019
**Founder & CEO:** Jan Oberhauser
**Industry:** Workflow Automation, AI-powered Enterprise Software
**Company Type:** Private (VC-backed)

### Company Expansion
- Opening new offices in New York and London in 2025
- Grew from 37 to 71 team members during 2024
- Currently has approximately 67-person team as of mid-2025

## Main Product & Key Milestones

### Product Overview

**n8n** is a fair-code workflow automation platform that combines visual workflow building with custom code capabilities, native AI integration, and offers both self-hosted and cloud deployment options. The platform features 400+ integrations.

### Core Product Features

1. **Visual Workflow Builder**
   - Drag-and-drop interface for creating automation workflows
   - Support for complex logic with conditional branches and loops
   - Custom code integration (JavaScript, Python)

2. **AI-Native Capabilities**
   - Native LangChain integration with ~70 AI-dedicated nodes
   - AI Workflow Builder (cloud users on v1.117.1+)
   - Custom AI agent creation
   - Multi-LLM support with full control over model and data flow

3. **Deployment Flexibility**
   - **Self-Hosted:** Docker, Kubernetes, air-gapped on-premise deployment
   - **Cloud:** Fully managed production-ready solution
   - Enterprise-grade security and compliance options

4. **Integrations**
   - 400+ native integrations
   - API connectivity for custom integrations
   - Webhooks and HTTP request nodes

## Main Competitors & Differentiation

### Competitive Landscape

| Competitor | Position | Key Strengths | Pricing | Target Market |
|------------|----------|---------------|---------|---------------|
| **Zapier** | Market Leader | 5,000+ integrations; easiest to use; plug-and-play | Premium pricing | Non-technical teams, SMBs |
| **Make** (Integromat) | Strong #2 | Visual diagrams; advanced branching; cost-effective | Competitive | Technical users, mid-market |
| **Workato** | Enterprise Leader | Enterprise iPaaS; robust governance; SLAs | $10K+/year | Large enterprises |
| **Tray.io** | Enterprise | Advanced data transformation; enterprise security | Enterprise pricing | Enterprise IT teams |
| **Pipedream** | Developer-focused | Code-first; serverless; free tier | Developer-friendly | Developers, startups |
| **Microsoft Power Automate** | Enterprise Suite | Office 365 integration; enterprise ecosystem | Microsoft 365 | Microsoft ecosystem users |
| **Node-RED** | Open Source | IoT focus; visual programming; free | Free (self-hosted) | IoT developers, hobbyists |

### n8n's Differentiation Strategy

#### 1. **Fair-Code Licensing Model**
- Open-source code with sustainable business model
- Free for self-hosting (non-commercial)
- Paid licenses for commercial self-hosting and cloud
- Community-driven development with enterprise support

#### 2. **AI-Native Platform**
- **70 AI nodes** with native LangChain integration
- Most advanced AI workflow capabilities among competitors
- AI Workflow Builder for automatic workflow generation
- Custom AI agent creation in single interface
- Multi-LLM support (OpenAI, Anthropic, Google, local models)

#### 3. **Self-Hosting + Cloud Flexibility**
- **Unique hybrid approach:** Choose deployment model
- Full data sovereignty for compliance-sensitive industries
- Air-gapped on-premise deployment option
- Docker and Kubernetes support
- No vendor lock-in

#### 4. **Developer-First Approach**
- Custom code nodes (JavaScript, Python)
- API-first architecture
- Git-based workflow versioning
- Extensive documentation and community support
- 400+ integrations with extensibility

#### 5. **Cost Advantage**
- **Self-hosted:** Free for non-commercial use
- **Cloud:** More affordable than Zapier for complex workflows
- Workflow-based pricing (not per-app)
- No hidden fees for premium connectors

#### 6. **Technical Sophistication**
- Advanced data transformation capabilities
- Complex conditional logic and loops
- Error handling and retry mechanisms
- Webhook support with custom authentication
- Database query nodes

### Competitive Positioning Matrix

**Ease of Use vs. Power:**
- **Easiest, Less Powerful:** Zapier
- **Balanced:** Make, Microsoft Power Automate
- **Most Powerful, Technical:** n8n, Pipedream, Workato

**Pricing:**
- **Free/Low Cost:** n8n (self-hosted), Pipedream, Node-RED
- **Mid-Range:** Make, n8n Cloud, Zapier (starter)
- **Enterprise:** Workato, Tray.io, Zapier (enterprise)

**AI Capabilities (2025):**
1. **n8n** - Most native AI integration
2. Make - Growing AI features
3. Zapier - Basic AI actions
4. Others - Limited AI support

### Why Customers Choose n8n Over Competitors

1. **Data Privacy & Security:** Self-hosting for regulated industries (healthcare, finance)
2. **Cost Savings:** No subscription fees for self-hosted; lower cloud costs for high-volume workflows
3. **AI Innovation:** Best-in-class AI workflow automation capabilities
4. **Flexibility:** Custom code + visual building hybrid approach
5. **No Vendor Lock-in:** Open-source foundation allows migration
6. **Community:** Active open-source community and contributor ecosystem
